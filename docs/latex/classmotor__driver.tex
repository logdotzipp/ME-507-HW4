\doxysubsection{Detailed Description}
Controls a DC motor using PWM and an HBridge Controller. 

The motor takes in a desired range of -\/127 to +127, which is then converted to be between -\/2000 and 2000 for use with the timer channel. The PWM signal is intended to be used with an H Bridge driver like those provided in ME 507 Lab.
