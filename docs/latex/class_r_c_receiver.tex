\doxysubsection{Detailed Description}
Measures the pulse width of a signal sent to the MCU by an RC receiver. 

The expected pulse width should be between 1000 and 2000us, centered at 1500us. This class is for use with a direct capture timer channel, capturing both rising and falling edges. Logic has been created to determine if the reading of rising/falling edges is in sync.
